## 2. 算法模型

采用图像退化模型：

```text
g(x, y) = f(x, y) * h(x, y) + n(x, y)
```

其中 `g(x, y)` 为降质图像，`f(x, y)` 为待复原图像，`h(x, y)` 为点扩散函数 PSF，`n(x, y)` 为加性噪声。

频域维纳滤波公式为：

```text
F_hat(u, v) = H*(u, v) / (|H(u, v)|^2 + NSR) * G(u, v)
```

其中：

- `G(u, v)` 是降质图像的傅里叶变换；
- `H(u, v)` 是运动模糊 PSF 的频率响应；
- `H*(u, v)` 是 `H(u, v)` 的共轭；
- `NSR` 是噪声功率谱与原图功率谱的比值，用于控制反卷积强度。

PSF 采用线性运动模糊模型，参数包括运动长度 `length` 和运动方向角 `theta`。为了减少 FFT 周期边界假设造成的边缘振铃，脚本在反卷积前对亮度图做 96 像素对称扩展，反卷积后裁剪回原尺寸。

## 3. 改进后的复原流程

最初只使用全图统一维纳反卷积时，树枝和天空区域会出现明显斜向振铃。为了让主要景物有更明显的复原效果，同时抑制高亮天空区域的伪影，最终方案采用以下流程：

1. 将彩色图像转换为亮度图，只对亮度层做维纳复原，避免 RGB 三通道分别反卷积造成颜色错位。
2. 使用运动模糊 PSF 对亮度图进行维纳反卷积。
3. 对亮度图先做边缘保持平滑，减少后续锐化放大的颗粒。
4. 将维纳反卷积结果作为保守的细节来源，而不是全强度替换原图。
5. 根据 Sobel 梯度构造结构区域掩膜：树干、地面、告示牌等有边缘纹理的区域使用更多复原细节；天空等高亮低纹理区域降低复原比例。
6. 对亮度层做受控锐化和局部对比度增强，使复原结果相较输入图有更清楚的视觉层次。
7. 对红色日期水印进行保护，避免反卷积破坏相机叠加文字。

最终参数：

```text
motion_psf_length    = 17
motion_psf_theta     = 40 degrees
NSR                  = 0.06
pad_size             = 96 pixels
luminance_blend      = 0.55
unsharp_amount       = 1.35
unsharp_radius       = 2.20
sharpen_threshold    = 0.016
local_contrast       = 0.20
```

其中 `luminance_blend` 和 `local_contrast` 会通过结构区域掩膜进行自适应调节：结构清晰的区域增强更强，高亮低纹理区域增强更弱。直接强反卷积会产生严重颗粒和振铃，因此最终结果采用“保守维纳细节 + 边缘保持平滑 + 受控锐化”的观感优先方案。

## 4. 参数优化过程

粗搜索范围：

```text
length: 13, 15, 17, 19, 21, 23, 25
theta : 30, 35, 40, 45, 50
NSR   : 0.02, 0.03, 0.04, 0.06, 0.08
```

粗搜索结果显示，较小的 `NSR=0.02` 可以显著提高 Tenengrad 和 Laplacian variance，但高亮天空、树冠和细枝区域会产生严重振铃。实验中也尝试了更强的 `NSR=0.035` 方案，虽然指标提升很大，但颗粒感过强，视觉上不自然。因此最终采用 `length=17, theta=40, NSR=0.06`，并只把维纳反卷积作为局部细节来源。

调参观察：

- `NSR` 太小：反卷积更强，边缘更锐，但树冠和天空振铃明显。
- `NSR` 太大：结果更平滑，但复原效果不明显。
- 只做统一亮度混合：主要景物提升有限，或者天空伪影过重。
- 加入结构区域掩膜后：树干、地面、告示牌等区域更清晰，天空区域的伪影相对可控。

## 5. 评价指标

由于没有清晰参考图，本实验采用无参考指标：

- `Tenengrad`：基于 Sobel 梯度能量的清晰度指标，越大表示边缘响应越强。
- `Laplacian variance`：二阶微分响应方差，过高可能表示锐化或噪声过强。
- `Entropy`：灰度信息熵，反映图像信息量。
- `High-frequency std`：高频成分标准差，用来观察高频增强和噪声放大。
- `Clip ratio`：像素越界比例，越低越好。

当前最终结果：

| 指标 | 输入图像 | 复原图像 | 比值 |
|---|---:|---:|---:|
| Tenengrad | 0.006046 | 0.017230 | 2.8499 |
| Laplacian variance | 0.011751 | 0.035473 | 3.0188 |
| Entropy | 7.263310 | 7.447161 | 1.0253 |
| High-frequency std | 0.027177 | 0.046713 | 1.7188 |
| Clip ratio | 0.000000 | 0.000000 | - |

结果分析：

- Tenengrad 提高约 185.0%，说明复原后边缘响应明显增强，树干、告示牌和地面纹理更容易辨认。
- Laplacian variance 提高约 201.9%，说明二阶边缘响应明显增强，复原图比输入图更锐。
- Entropy 提高约 2.5%，说明复原后图像灰度层次和局部信息量有所增加。
- High-frequency std 提高约 71.9%，说明增强后高频细节增加；由于最终方案加入边缘保持平滑和天空保护，视觉颗粒比强反卷积版本更可控。
- Clip ratio 为 0，说明没有明显截断溢出。

视觉上，复原图相较原图的树干边缘、路面纹理、右侧告示牌轮廓和前景树叶层次更清楚，整体灰雾感降低。需要说明的是，该降质图像没有清晰参考图，且模糊较重，单纯强反卷积会产生明显颗粒和振铃；因此最终结果不是完全恢复清晰原图，而是在复原模型基础上选择了观感更稳定的增强折中方案。
