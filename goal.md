## 3. 图像退化模型

图像复原通常把降质过程表示为：

```text
g(x,y) = f(x,y) * h(x,y) + n(x,y)
```

其中：

1. `f(x,y)` 是理想清晰图像。
2. `g(x,y)` 是当前观测到的降质图像，即 `blurred wood.bmp`。
3. `h(x,y)` 是空间域退化函数，也叫点扩散函数 PSF。
4. `n(x,y)` 是加性噪声。
5. `*` 表示卷积。

在频率域中，上式写成：

```text
G(u,v) = F(u,v)H(u,v) + N(u,v)
```

复原的目标是在只知道 `g(x,y)`、不知道真实 `f(x,y)` 的情况下，估计 `H(u,v)`，然后从 `G(u,v)` 中尽可能恢复 `F(u,v)`。

如果直接使用逆滤波：

```text
F_hat(u,v) = G(u,v) / H(u,v)
```

当 `H(u,v)` 在某些高频位置接近 0 时，噪声会被严重放大，图像中容易出现振铃、颗粒和伪影。因此本任务不把直接逆滤波作为最终方法，而是把维纳滤波作为主复原方法。

## 4. 主算法选择

本任务采用的主流程为：

```text
读取降质图像
-> 灰度或亮度通道预处理
-> 频谱分析
-> 预估退化函数 PSF
-> 维纳滤波复原
-> 后处理抑制伪影
-> 无参考指标评价
-> 保存复原结果和分析图
```

与 `image.png` 的对应关系应写清楚：

```text
输入降质图像 g(x,y)
-> 亮度/灰度通道预处理
-> FFT 得到 G(u,v)
-> 估计退化函数 H(u,v)，并构造维纳复原滤波器 W(u,v)
-> F_hat(u,v) = W(u,v)G(u,v)
-> IFFT 得到复原亮度图 f_hat(x,y)
-> 后处理与颜色重建
-> 输出复原图像
```

因此，`image.png` 在本任务中作为算法框图样式和频域处理思路参考。若直接照图中的 `ln` 和 `exp` 做同态滤波，应作为“光照不均复原”的备选或对照方法；当前木纹图像的主问题是模糊，核心仍应是退化函数估计和维纳去卷积。

选择该流程的理由如下：

1. `blurred wood.bmp` 的主要问题是模糊，高频细节衰减明显，符合图像复原中的退化模型。
2. 退化函数未知，需要先通过频谱和参数实验估计 PSF。
3. 直接逆滤波对噪声敏感，容易放大高频噪声。
4. 维纳滤波能同时考虑退化函数和噪声影响，更适合未知噪声条件下的图像复原。
5. 木纹图像包含大量纹理，复原后容易出现振铃，因此需要结合无参考指标和视觉观察选择参数，不能只追求锐度数值最大。

## 5. 退化函数预估思路

由于没有清晰参考图，退化函数不能直接由 `H(u,v)=G(u,v)/F(u,v)` 得到，只能估计。预估过程分为频谱观察、候选 PSF 构造、参数搜索和结果筛选四步。

### 5.1 频谱观察

首先对降质图像进行二维傅里叶变换：

```matlab
F = fft2(grayImg);
F_shift = fftshift(F);
spectrum = log(1 + abs(F_shift));
```

观察频谱时重点关注：

1. 中心低频是否明显集中。
2. 高频部分是否整体较暗。
3. 是否存在运动模糊常见的方向性暗线或条纹。
4. 是否存在周期噪声峰值。
5. 是否存在明显光照不均导致的低频能量异常。

如果频谱中高频整体衰减，但没有明显周期噪声峰值，则说明问题更接近普通模糊，而不是周期噪声污染。

### 5.2 候选退化函数

本任务优先考虑两类退化函数。

第一类是运动模糊 PSF：

```matlab
psf = fspecial('motion', LEN, THETA);
```

其中：

1. `LEN` 表示运动模糊长度。
2. `THETA` 表示运动方向角度。
3. 适合相机抖动、物体运动或扫描方向模糊造成的退化。

第二类是高斯模糊 PSF：

```matlab
psf = fspecial('gaussian', hsize, sigma);
```

其中：

1. `hsize` 表示滤波器尺寸。
2. `sigma` 表示模糊程度。
3. 适合离焦、成像系统低通效应或均匀软化造成的退化。

如果运动模糊复原结果出现明显方向性振铃，而高斯 PSF 的结果更自然，则最终选择高斯 PSF；反之，如果运动 PSF 能明显恢复木纹方向细节且振铃可控，则优先选择运动 PSF。

### 5.3 参数搜索范围

建议在 `code2.m` 中设置如下参数范围：

```matlab
motionLens = 5:2:35;
motionAngles = -90:5:90;
gaussianSigmas = 0.8:0.2:4.0;
nsrList = [1e-5, 3e-5, 1e-4, 3e-4, 1e-3, 3e-3, 1e-2];
```

每一组候选参数都执行一次维纳滤波，然后计算无参考指标。最终参数不应机械选择指标最大值，而应结合以下原则：

1. 木纹和边缘比原图更清晰。
2. 背景噪声没有明显放大。
3. 没有大面积振铃波纹。
4. 没有过度锐化导致的白边或黑边。
5. 指标提升稳定，不是局部异常。

### 5.4 最终退化函数选择原则

最终建议优先选择“视觉自然 + 清晰度提升明显 + 伪影较少”的参数，而不是单纯选择拉普拉斯方差最大的参数。

原因是图像复原中的锐化和噪声放大经常同时发生。如果只最大化清晰度指标，可能得到一张噪声很强、振铃明显、但数值指标较高的图像。这种结果不适合作为课程设计报告中的最终复原结果。

## 6. 维纳滤波复原

维纳滤波的频率域表达可写为：

```text
F_hat(u,v) = [H*(u,v) / (|H(u,v)|^2 + K)] G(u,v)
```

其中：

1. `H*(u,v)` 是 `H(u,v)` 的共轭。
2. `K` 是噪声功率谱与原图功率谱之比的近似值，在 MATLAB 的 `deconvwnr` 中对应 `NSR`。
3. `K` 越小，复原越激进，细节更强，但噪声和振铃也更明显。
4. `K` 越大，复原越保守，噪声较少，但图像可能仍然偏模糊。

MATLAB 中的核心调用方式为：

```matlab
restored = deconvwnr(grayImg, psf, nsr);
```

本任务使用维纳滤波作为最终复原方法，是因为它比逆滤波更稳健，能在高频细节恢复和噪声抑制之间取得折中。

## 7. 具体处理流程

### 7.1 读取和预处理

1. 使用 `imread` 读取 `原图与参考图\blurred wood.bmp`。
2. 如果是 RGB 图像，保留 RGB 图像用于最终展示，同时转换为灰度图或亮度通道用于复原实验。
3. 使用 `im2double` 转换到 `[0,1]` 范围，避免计算过程中发生整数截断。
4. 保存输入图像副本到 `处理后图像\task2_input_blurred.bmp`。

建议代码结构：

```matlab
inputImg = imread('原图与参考图\blurred wood.bmp');
inputDouble = im2double(inputImg);

if size(inputDouble, 3) == 3
    grayImg = rgb2gray(inputDouble);
else
    grayImg = inputDouble;
end
```

### 7.2 频谱分析

1. 对灰度图做 `fft2`。
2. 用 `fftshift` 将低频移动到频谱中心。
3. 用 `log(1 + abs(...))` 压缩动态范围，得到便于观察的频谱图。
4. 保存频谱图到 `处理后图像\task2_degradation_analysis.png`。
5. 根据频谱判断是否存在方向性模糊、普通低通模糊或周期噪声。

### 7.3 候选 PSF 复原实验

1. 遍历运动模糊参数 `LEN` 和 `THETA`。
2. 遍历高斯模糊参数 `sigma`。
3. 对每个 PSF 遍历多个 `NSR`。
4. 使用 `deconvwnr` 得到候选复原图。
5. 对候选复原图进行裁剪，确保像素值在 `[0,1]` 范围内。
6. 计算无参考指标。
7. 把每组参数和指标保存成表格，建议文件名为 `处理后图像\task2_degradation_candidates.csv`。

候选实验的目标不是一次就找到绝对最优参数，而是形成有依据的参数比较过程，便于报告中说明“如何预估退化函数”。

### 7.4 后处理

维纳滤波后可能出现轻微噪声或振铃。后处理必须保持克制，避免把复原问题变成单纯锐化问题。

建议后处理流程如下：

1. 使用 `mat2gray` 或 `min(max(img,0),1)` 限制灰度范围。
2. 若噪声明显，可使用轻微中值滤波：

   ```matlab
   restoredDenoised = medfilt2(restored, [3 3]);
   ```

3. 若边缘发虚，可使用温和反锐化：

   ```matlab
   blurSmall = imgaussfilt(restored, 0.6);
   finalImg = restored + 0.25 * (restored - blurSmall);
   finalImg = min(max(finalImg, 0), 1);
   ```

4. 如果温和锐化引入明显白边或噪声，应降低锐化系数或取消锐化。

### 7.5 彩色结果处理

如果只对灰度图进行复原，则最终结果可以保存为灰度图。为了保持原图的彩色观感，也可以采用亮度通道复原方式：

1. 将 RGB 图像转换到 YCbCr。
2. 只对 Y 亮度通道进行退化函数估计和维纳复原。
3. 保留 Cb、Cr 色度通道。
4. 合并复原后的 Y 通道与原 Cb、Cr 通道，再转回 RGB。

该方式能避免分别对 RGB 三通道复原造成颜色偏移，同时保留木纹颜色。

建议最终报告中说明：主要复原对象是亮度细节，因此优先处理灰度/亮度通道。

## 8. 质量评价方法

由于当前没有 `blurred wood.bmp` 对应的清晰参考图，不能直接计算严格意义上的 MSE、SNR、PSNR 和 SSIM。全参考指标需要真实清晰图像作为比较对象，否则数值没有意义。

因此，本任务采用无参考评价和视觉评价相结合的方式。

### 8.1 拉普拉斯方差

拉普拉斯方差用于衡量图像边缘和高频细节强度：

```matlab
lapKernel = fspecial('laplacian', 0.2);
lapImg = imfilter(grayImg, lapKernel, 'replicate');
lapVar = var(lapImg(:));
```

复原后拉普拉斯方差适度上升，说明边缘和纹理更清晰；如果上升过大且视觉上噪声明显，说明可能过度复原。

### 8.2 Tenengrad 梯度能量

Tenengrad 指标基于 Sobel 梯度：

```matlab
gx = imfilter(grayImg, fspecial('sobel')' , 'replicate');
gy = imfilter(grayImg, fspecial('sobel')  , 'replicate');
tenengrad = mean(gx(:).^2 + gy(:).^2);
```

该指标越高，通常表示图像边缘越清楚。但它同样会受噪声影响，因此必须结合视觉评价。

### 8.3 高频能量比例

在频率域中计算高频区域能量占比：

1. 对图像做 FFT。
2. 以频谱中心为圆心构造半径阈值。
3. 统计高频区域能量与总能量之比。

复原后高频能量比例应有所提升，说明被模糊抑制的纹理细节得到恢复。

### 8.4 视觉评价

视觉评价重点包括：

1. 木纹纹理是否比输入图更清楚。
2. 树干边界是否更锐利。
3. 是否存在明显振铃。
4. 是否存在噪声颗粒被放大的现象。
5. 是否出现过曝、过暗或颜色偏移。
6. 整体观感是否自然。

### 8.5 如果后续获得清晰参考图

如果后续补充了清晰参考图，则应增加全参考指标：

```matlab
mseVal = mean((refImg(:) - restoredImg(:)).^2);
psnrVal = 10 * log10(1 / mseVal);
snrVal = 10 * log10(mean(refImg(:).^2) / mseVal);
ssimVal = ssim(restoredImg, refImg);
```

报告中应明确区分：

1. 无参考指标用于当前无清晰参考图的情况。
2. MSE、SNR、PSNR、SSIM 只在存在真实清晰参考图时使用。

5. 对每个候选结果计算：
   - 拉普拉斯方差。
   - Tenengrad 梯度能量。
   - 高频能量比例。
   - 灰度均值。
   - 灰度标准差。
6. 记录候选结果表。

### 10.5 选择最终参数

参数选择建议遵循以下规则：

1. 先排除振铃和噪声明显的候选。
2. 在剩余候选中选择 Tenengrad 和拉普拉斯方差比输入图有明显提升的结果。
3. 若多组候选接近，选择 `NSR` 较大的保守结果，避免噪声放大。
4. 最终输出选择理由，例如：

   ```text
   选择 motion PSF，LEN=15，THETA=0，NSR=0.001。
   该参数下木纹细节增强明显，振铃较轻，清晰度指标较输入图提升。
   ```

具体数值应以实际运行结果为准，不能在没有实验前写死。

### 10.6 最终复原与保存

1. 用最终 PSF 和 NSR 重新执行维纳滤波。
2. 保存 `task2_wiener_only.bmp`。

4. 保存 `task2_final_result.bmp`。
