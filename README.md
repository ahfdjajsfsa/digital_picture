# 数字图像处理实践——课程设计 3

本项目的目标是完成并回答《数字图像处理实践》课程设计 3（图像增强与复原算法综合应用）PDF 中提出的任务要求，围绕图像增强与图像复原两个方向实现算法设计、程序验证与结果分析。

## 任务内容

### 1. 图像增强算法设计
设计一种**空域与频域结合**的图像增强算法，对指定带噪图像进行处理，目标是去除噪声、提升图像质量。

要求如下：
- 噪声由随机噪声和周期噪声混合构成；
- 处理带噪图像，清晰图像仅作为算法评估使用；
- 去噪后，使用**均方误差（MSE）**、**信噪比（SNR）**、**结构相似度（SSIM）**等图像质量指标评估算法性能；
- 撰写完整的科技报告，说明算法设计、实现过程与评估结果。

图像选择规则：
- 根据自己的学号**尾数**选择图像；
- **奇数**选择第一组，**偶数**选择第二组。

### 2. 图像复原算法设计
针对一幅**降质图像**设计复原算法，完成图像复原并进行性能评估。

要求如下：
- 对给定的降质图像设计复原算法并编程实现；
- 使用图像质量指标评估复原效果。

图像选择规则：
- 根据自己的学号**尾数**选择图像；
- **奇数**选择图 2，**偶数**选择图 1。

### 一、 空间域图像增强算法

**1. 点操作类**
*   **灰度变换**：包含线性变换（全域、截取式、分段线性变换、图像反色）、对数变换、指数变换、位平面切片
*   **直方图处理**
*   **算术与逻辑运算**

**2. 模板操作类（空域滤波）**
*   **空域平滑算法**：算术均值滤波器、高斯滤波器、双边滤波器
*   **空域锐化算法（一阶与二阶微分算子）**：Roberts梯度算子、Prewitt梯度算子、Sobel梯度算子、Laplacian（拉普拉斯）算子、LoG（高斯拉普拉斯）算子
